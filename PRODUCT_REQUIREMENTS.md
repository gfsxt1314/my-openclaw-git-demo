# 五子棋游戏 - 产品需求文档

## 项目概述
一个网页版五子棋游戏，支持双人对战（本地同屏）。

## 功能需求

### MVP（最小可行产品）
1. **游戏界面**
   - 15×15 标准棋盘
   - 黑白棋子交替落子
   - 清晰显示当前执子方

2. **游戏规则**
   - 黑棋先行
   - 五子连珠获胜（横、竖、斜）
   - 禁止在已有棋子位置落子
   - 判断胜负

3. **游戏控制**
   - 重新开始游戏
   - 显示游戏状态（进行中/黑胜/白胜）

### 后续迭代（可选）
- [ ] 悔棋功能
- [ ] 游戏计时
- [ ] 人机对战
- [ ] 在线多人对战
