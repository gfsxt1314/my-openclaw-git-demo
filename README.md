# 五子棋游戏 🏀

一个精美的网页版五子棋游戏，支持双人对战（本地同屏）。

## 🎮 在线试玩

[GitHub Pages 链接](https://gfsxt1314.github.io/my-openclaw-git-demo/) (部署后)

## ✨ 功能特性

- 🎯 15×15 标准棋盘
- ⚫⚪ 黑白棋子交替落子
- 🏆 自动判断胜负（横、竖、斜五子连珠）
- 🔄 一键重新开始
- 📱 响应式设计，支持移动端
- 🎨 精美的渐变 UI

## 🚀 快速开始

### 本地运行

```bash
# 克隆项目
git clone https://github.com/gfsxt1314/my-openclaw-git-demo.git

# 进入目录
cd my-openclaw-git-demo

# 直接用浏览器打开 index.html
open index.html
```

### 部署到 GitHub Pages

```bash
# 启用 GitHub Pages
gh repo edit --enable-pages

# 或者在 GitHub 仓库设置中手动开启
```

## 🎯 游戏规则

1. 黑棋先行，双方交替落子
2. 先形成五子连珠（横、竖、斜）者获胜
3. 不能在有棋子的位置落子
