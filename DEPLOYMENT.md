### 如何更新

每次推送到 `main` 分支后，GitHub Pages 会自动重新构建：

```bash
# 修改代码后
git add .
git commit -m "更新内容"
git push
```

等待 1-2 分钟，新版本就会自动上线！

### 自定义域名（可选）

如果想用自定义域名：

1. 在 GitHub 仓库 Settings → Pages 中添加自定义域名
2. 在 DNS 提供商处添加 CNAME 记录
